**#Student Performance Analysis**



**#Project Overview**

This project analyzes student exam scores using Python and visualizes performance using data analysis techniques.



**#Dataset**

The dataset contains student marks for:

\- Math

\- Science

\- English



**#Technologies Used**

\- Python

\- Pandas

\- Matplotlib

\- Jupyter Notebook



**#Analysis Performed**

\- Calculated average scores

\- Visualized score distribution using bar chart

\- Data frame creation and processing
